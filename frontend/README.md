### Development
For development go inside `frontend` directory and install dependencies terminal:

```bash
$ cd frontend
$ npm install
$ npm run start
```

### Production build
In terminal run commands:
```bash
$ npm run build
```

### Production build for backend  
 Production build and move built to backend folder
```bash 
$ npm run build_local
```

### Run tests:
```bash
$ npm run test
```
